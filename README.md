# SOR-Project
Proyecto de la asignatura "Sistemas operativos en red"

##Descripción del proyecto
Gestor de subastas online para la compra y venta de obras de arte. Se proponen diversos clientes para el acceso al mismo sistema, utilizando diferentes tecnologías.
- Cliente en .NET(C#)
- Cliente en JAVA
- Cliente Web (ASP)

La comunicación entre clientes y gestor se realiza mediante el usdo de ActiveMQ, WS y la ayuda de Juddi.
